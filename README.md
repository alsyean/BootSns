Spring Boot를 이용한 Simple SNS
================================

### 작품 개요
Spring Boot를 이용한 간단한 SNS
* * *

### 개발 환경
* Spring Boot
* Eclipse
* JDK 1.8
* Maven 3
* JPA
* MySQL 8.0
* Freemarker
* * *

### 스토리
* 회원 가입 
* 이메일 인증
* 로그인/로그아웃
* 포스팅
* 전체 포스팅 보기
* 팔로우 언팔로우
* 팔로우 및 내 글 보기
* 상세보기
* 내 글 수정 및 삭제 
* 게시글 5개씩 더 보기
* 비회원도 댓글 작성
* 비밀번호로 댓글 삭제 가능
